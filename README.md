# CodeAlpha_Unemployment_Analysis


## 📌 Objective
To analyze unemployment trends before and during the COVID-19 period using data visualization techniques.

---

## 📂 Datasets Used
Two datasets were used:

1. Dataset 1:
   - Contains unemployment data before COVID-19
   - Columns: Region, Date, Frequency, Estimated Unemployment Rate, Area

2. Dataset 2:
   - Contains unemployment data during COVID-19
   - Columns: Region, Date, Estimated Unemployment Rate, Estimated Employed, Latitude, Longitude

---

## 🛠️ Tools & Libraries
- Python
- Numpy
- Pandas
- Matplotlib
- Seaborn

---

## 🔍 Data Preprocessing
- Handled missing values in Dataset 1 by removing null rows
- Converted Date column into datetime format
- Cleaned column names for easy analysis

---

## 📊 Analysis Performed

### 🔹 Dataset 1 (Before COVID)
- Unemployment trend over time
- Region-wise comparison

### 🔹 Dataset 2 (During COVID)
- COVID-19 unemployment trend
- Top regions with highest unemployment
- Monthly unemployment pattern

---

## 📈 Key Insights
- Unemployment was relatively stable before COVID-19
- A sharp increase was observed during the pandemic
- Significant regional differences exist
- Monthly variations indicate changing economic conditions

---

## 📝 Conclusion
Unemployment rates increased significantly during the COVID-19 period compared to pre-COVID levels. The impact varied across regions and months, highlighting the uneven economic effects of the pandemic.
